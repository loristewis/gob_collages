# gob_collages

Empty project.

## Building and running on localhost

First install dependencies:

```sh
npm install
```

To create a production build:

```sh
npm run build-prod
```

To create a development build:

```sh
npm run build-dev
```

To launch a development server: (Project will be running at http://localhost:8090/)

```sh
npm run serve
```
